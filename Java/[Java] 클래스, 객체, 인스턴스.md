# [Java] 클래스, 객체, 인스턴스

![image](https://github.com/user-attachments/assets/183068c6-72e3-4b0e-a78f-63bc826675ec)

- 클래스(Class)는 객체를 만들어 내기 위한 `설계도` (= 붕어빵 틀) 
- 객체(Object)는 `구현할 대상` (= 붕어빵)
- 인스턴스(Instance)는 특정 클래스로부터 생성된 객체
    - 클래스의 타입으로 선언되면 `객체`, 그 객체가 메모리에 할당되어 실제로 사용되면 `인스턴스` 
    - 객체가 `어떤 클래스로부터 만들어졌는지`, `어떤 클래스에 속해있는지` 강조하고 싶을 때 `인스턴스`라는 용오를 사용함.
        - ex. student1은 Student클래스의 인스턴스!
        - 보통 student1은 Student의 객체이다. 라고 말하는 대신 student1은 Student의 인스턴스라고 특정 클래스와의 관계를 명확히 할 때 인스턴스라는 용어를 주로 사용
    - (근데 보통 객체, 인스턴스 용어는 혼용해서 씀~!)
