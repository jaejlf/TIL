- `이슈` 발행해서 TIL 작성
- 잘 이해안되는 것, 추가로 봐야할 내용은 `이슈 코멘트`로 남기기
- 이슈 `close`할 때, 처리되지 않은 코멘트가 있다면 [unchecked](https://github.com/jaejlf/TIL/issues?q=is%3Aissue+is%3Aclosed+label%3Aunchecked+) 라벨링
- 주(week) 단위로 발행된 이슈들을 모아서 `Pull Request`
- [unchecked](https://github.com/jaejlf/TIL/issues?q=is%3Aissue+is%3Aclosed+label%3Aunchecked+) 코멘트는 `한 달 이내` 처리하기. (그럼에도 남은 것이 있다면 `새로운 이슈`로 관리)
