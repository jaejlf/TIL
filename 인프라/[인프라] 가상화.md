# 가상화

![image](https://github.com/user-attachments/assets/98246068-dfd2-483d-98af-b011b950666f)

- 실체가 없는 것을 마치 존재하는 것처럼 보이게 하는 기술
    - 물리 서버 1대에 가상 서버를 여러 대 구축하거나, 물리 서버 여러 대에 가상 서버 1대를 구축하거나, …
- 가상화의 유연성 : 물리적 환경의 제약을 받지 않고, 유연하게 환경을 구축할 수 있다. (ex. 물리 서버의 OS가 Window → VirtualBox(가상화 소프트웨어) → 가상 서버에서 CentOS를 실행)

<br>

# 클라우드

- 가상화 기술을 기반으로 구성된 서비스
- 가상화 기술로 구축된 인프라 또는 그 위에 구축된 서비스
- ex. 아마존 웹 서비스(AWS), 구글 클라우드 플랫폼(GCP)

<br>

### SaaS(Software as a Service)

- `소프트웨어`로 제공되는 클라우드 서비스
- 이미 완성된 제품의 서비스

<br>

### PaaS(Platform as a Service)

- `플랫폼`으로 제공되는 클라우드 서비스
- 완성된 서비스 X, 애플리케이션을 구축하기 위해 필요한 스토리지 등을 제공 (사용자가 스스로 애플리케이션을 구축해야함.)

<br>

### IaaS(Infrastructure as a Service)

- `인프라`로 제공되는 클라우드 서비스
- 데이터베이스나 인터넷 등 환경 구축까지 사용자가 해야함.
